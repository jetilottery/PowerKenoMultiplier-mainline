Starship Keno Multiplier

Explorez les richesses interstellaires dans le jeu Starship Keno Multiplier! Choisissez jusqu’à 10 cases Keno. Au début de la partie, six cases comprenant des multiplicateurs sont également ajoutées à la grille Le jeu sélectionne 20 numéros Keno. Si les numéros sélectionnés correspondent à vos numéros, vous gagnez un lot selon la table de paiement! Si deux cases comprenant des multiplicateurs ou plus correspondent aux numéros que vous avez sélectionnés, le lot accordé est multiplié jusqu’à 10 fois!

Comment miser

Cases

Affiche le nombre actuel de cases à sélectionner.

Appuyez sur la flèche gauche (–) pour diminuer le nombre de cases à sélectionner.

Appuyez sur la flèche droite (+) pour augmenter le nombre de cases à sélectionner.

Coût du billet

Affiche le coût actuel du billet.

Appuyez sur la flèche gauche (–) pour diminuer le coût du billet.

Appuyez sur la flèche droite (+) pour augmenter le coût du billet.

Appuyez sur ACHETER pour débiter le prix du billet de votre solde et commencer la partie.

Comment jouer

Le but du jeu Starship Keno Multiplier est d’obtenir des numéros Keno identiques aux numéros sélectionnés par le joueur ou aux numéros comprenant des multiplicateurs.

Sélectionnez des cases à numéro dans la grille de jeu ou appuyez sur SÉLECTION RAPIDE pour sélectionner des numéros automatiquement. Appuyez sur JOUER. Les cases comprenant des multiplicateurs sont ajoutées au hasard à la grille. Lorsque toutes les sélections ont été effectuées, 20 numéros Keno sont tirés, et le jeu indique les numéros tirés sur la grille de jeu.

Si les numéros que vous avez sélectionnés sont identiques aux numéros Keno tirés, vous gagnez le lot correspondant dans la table de paiement. Lorsque deux cases comprenant des multiplicateurs ou plus correspondent aux numéros que vous avez sélectionnés, le lot accordé est multiplié par la valeur du multiplicateur.

Le jeu prend fin lorsque tous les numéros tirés sont indiqués sur la grille. Le jeu affiche un message indiquant le résultat de la partie.

Vitesse

Appuyez sur VITESSE pour changer la vitesse à laquelle les numéros Keno sont tirés. Vous avez trois options de vitesse.

Sélection rapide

Appuyez sur SÉLECTION RAPIDE pour sélectionner des numéros automatiquement. La partie continue jusqu’à ce que toutes les sélections aient été effectuées.

Annuler tout

Appuyez sur ANNULER TOUT pour annuler toutes les sélections.

Règles

Toute utilisation inappropriée et toute défectuosité annulent jeux et paiements.

Les gains sont affichés en argent.

Les choix effectués par le joueur pendant le jeu n’ont aucune incidence sur le résultat de la partie.
